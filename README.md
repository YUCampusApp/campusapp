# Yeditepe Campus App

Yeditepe Campus App is an integrated student life management application designed for Yeditepe University students and instructors. The project aims to bring academic services, campus services, transportation information, and daily student needs into a single centralized mobile platform.

The application was developed as part of the CSE 344 / ISE 402 Interdisciplinary Project course at Yeditepe University.

## Project Purpose

Campus-related services at Yeditepe University are currently distributed across different platforms or handled manually. This creates problems such as time loss, lack of real-time information, long queues, and difficulty accessing campus resources.

Yeditepe Campus App aims to solve these problems by providing a user-friendly and centralized system where users can manage their academic and campus life more efficiently.

## Main Features

### Authentication and User Management
- User login with university credentials
- Secure logout
- Profile viewing and editing
- Password change and reset support
- Notification and privacy preference management

### Dashboard
- Personalized welcome message
- Today’s classes and next course information
- Weather summary
- Campus announcements
- Favorite modules for quick access
- Bottom navigation bar for easy movement between pages

### Academic Services
- View academic information such as GPA, department, advisor, and credits
- View daily course schedule
- Upload lecture notes
- Browse and download shared lecture notes
- Search notes by course name or keyword
- Build course schedules with conflict detection
- Generate alternative course schedules

### Campus Services
- Library reservation system
- View library occupancy and available seats
- Cafeteria menu display
- Campus hairdresser appointment booking
- Market and stationery stock tracking
- Shuttle arrival time tracking
- Campus map and building finder

### Notifications
- Reservation confirmations
- Appointment reminders
- Upcoming class notifications
- Important campus announcements

## Target Users

The application is designed mainly for:

- Yeditepe University students
- Yeditepe University instructors
- System administrators
- Campus service staff

Students and instructors use the app to access academic and campus services, while administrators and service staff update menus, stock information, shuttle data, reservations, and other operational content.
